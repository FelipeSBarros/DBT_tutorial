# Setting up `dbt` cloud

Learning Objectives:
- Load training data into your data platform
- Set up an empty repository and connect your GitHub account to dbt Cloud.
- Set up your warehouse and repository connections.
- Navigate the dbt Cloud IDE.
- Complete a simple development workflow in the dbt Cloud IDE.

PAra realizar a configuração do `dbt cloud` é neceessário criar/configurar a plataforma de dados a ser usada.
Considerando o tutorial, vou seguir com `snowflake`:

> In our demos, we will be using Snowflake as our data warehouse. You will need access to your own data platform to complete the practice exercises.
